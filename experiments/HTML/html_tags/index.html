Experiment: First HTML

I. What is HTML

HTML is a markup language for describing web documents (web pages).

  • HTML stands for Hyper Text Markup Language
  • A markup language is a set of markup tags
  • HTML documents are described by HTML tags
  • Each HTML tag describes different document contentThis experiment is about HTML

HTML Example

                <!DOCTYPE html>
                <html>
                    <body>
                        <h1>First Heading</h1>
                        <h2>Second Heading</h2>
                        <p>Paragraphy/p>
                        <a href="www.w3schools.com">Markup snippet for a link to </a>
                    </body>
                </html>

Explain for Example

  • The DOCTYPE declaration defines the document type
  • The text between <html> and </html> describes the web document
  • The text between <body> and </body> describes the visible page content
  • The text between <h1> and </h1> describes a heading
  • The text between <p> and </p> describes paragraph
Using the description, a web browser can display a document with a heading and a paragraph.

II. HTML Tags

Raw text elements are constructed with 1) a start tag <tag>, 2) some amount of text content, 3) and end tag </tag>

  • <title>...</title>
    Define a document title. For example, the title of this page is "Experiment: HTML".
    <title>Experiment: HTML</title>
  • <p>...</p>
    Creates a paragraph of some text. For example, the following code
    <p>Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.</p>
    will generate the following content

    Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.

  • <h1>...</h1> <h2>...</h2> ... <h6>...</h6>
    Section headings at different levels. <h1> delimits the highest-level heading, <h2> the next level down (sub-section), <h3> for a level below that, and so on to <h6>. They are sometimes referred to collectively as <hn> tags, n meaning any of the available heading levels.

    This is h1

    This is h2

    This is h3

    This is h4

    This is h5
    This is h6
  • <ul>...</ul> , <ul>...</ul> and <li>...</li>
    <ol> is an ordered (enumerated) list. <ul> is an unordered (bulleted list). <li> is a list item in ordered or unordered list
    This following code is for ordered list:
    <ol>
        <li>Decide how much rice you will need</li>
        <li>Gather your supplies</li>
        <li>Cooking the rice</li>
    </ol>
    it will generate:
    1. Decide how much rice you will need
    2. Gather your supplies
    3. Cooking the rice
    This following code is for unordered list:
    <ul>
        <li>What is HTML</li>
        <li>What is CSS</li>
        <li>What is ASP.NET</li>
    </ul>
    it will generate:
    • What is HTML
    • What is CSS
    • What is ASP.NET
  • <a>...</a>
    An anchor element is called an anchor because web designers can use it to anchor a URL to some text on a web page. When users view the web page in a browser, they can click the text to activate the link and visit the page whose URL is in the link
    The following code is an example of anchor tag:
    <a href="http://net4.ccs.neu.edu/home/rheaxu/">an link to my home page</a>
    It will generate the following link to the home page of my website:
    an link to my home page
