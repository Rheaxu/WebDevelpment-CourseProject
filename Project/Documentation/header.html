Page Header

This Documentation is about the header of the website, which is consistent across the whole website.

See the header in the home page

1. Overview

This header is based on ASP.NET loginview control, it displays different views for loggedin and non-loggedin users.

The header contains:

  • Navigation bar
  • If you are not logged in: a "log in" link and a "register" link
  • If you are logged in: a "Welcome" label and a link to your profile

2. Navigation Bar:

1. Contents:

  • Home: Navigate to Home Page
  • Sheet Music:Navigate to All Members' Sheet Music Collection page
  • Photo:Navigate to All Members' Photo Gallery page
  • My Profile (for logged in user):Navigate to My Profile page
  • Search:Navigate to Search page
  • Login status (Login/Logouast):Navigate to Login page

2. Jquery transform

The rotation of each option is implemented by using jquery transform and the css provided by jquery. The main part of the css is:
