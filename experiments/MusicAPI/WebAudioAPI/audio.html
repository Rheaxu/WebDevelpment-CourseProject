Experiment: Web Audio API

This experiment is related to my project
Comparing to the Audio tag in HTML, The Web Audio API provides a powerful and versatile system for controlling audio on the Web, allowing developers to choose audio sources, add effects to audio, create audio visualizations, apply spatial effects (such as panning) and much more.

Details of the experiments:

There are two ways to create Audio.
1. Web Audio Element
By this way, we create the audio by <audo>tag
Press "1","2","3" on your keyboard, you will hear three different notes
2. Web Audio API
By this way, we create the audio by using the audio API. We define the Sound as an object and use it to create different notes.
Press "a","s","d","f","g","h","j" on your keyboard, you will hear "la","ti" "do","re","mi","fa","so"
