Experiment: Express Request

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.
This experiments practices on how to use experss request and response module.

1. res.send([body]) ---- Send a response
In the following code, the server send back a response "This is a plain response sent from server" in a plain format. 

app.get('/res_sendpalinres', function (req, res) {
    res.send("This is a plain response sent from server");
});
Response sends palin text
2.res.json([body]) ---- Send a response in json format In the following code, the server send back a response 
app.get('/res_sendjsonres', function (req, res) {
    res.json({name:"message"});
});
Response sends json text
3.res.redirect([status],url) ---- Redirect to another page The following code redirect the page to google main page 
app.get('/res_redirecttogoogle', function (req, res) {
    res.redirect('http://www.google.com');
});
Response Redirect the page to google
4.res.download(path,[filename],[fn]) ---- Download a file The following code downloads a file when you go to the page 
app.get('/res_download', function (req, res) {
    res.download('../../images/qmark.jpg');
});
Response download a file
