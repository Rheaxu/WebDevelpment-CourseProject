Experiment: Hello from JQeury

This is the first trial on JQuery
jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Details of the experiments:

1. Download JQuery from its website,
2. Click the button to get greeting from jquery,the <span>will be appended to the button

The javascript for this is 

$(document).ready(function () {
    $("button.hello").click(function () {
        console.log("Append 'Hello from jquery after the button'")
        $("span.jquery_hello").append("Hello from jquery")
    });
});
