Experiment: Angularjs

I. What is Angular

AngularJS is an open-source web application framework, maintained by Google and community, that assists with creating single-page applications, which consist of one HTML page with CSS, and JavaScript on the client side. Its goal is to simplify both development and testing of web applications by providing client-side model–view–controller (MVC) capability as well as providing structure for the entire development process, from design through testing.

Example of how to use angularjs

                <div ng-controller="HelloWorldController">
                <Wh2>Msg from hello world controller</Wh2>
                {{message}}
                <div>

                <script>
                    function HelloWorldController($scope) {
                    $scope.message = "Hello World";
                    console.log("Hello World")
                }
                <script>

The code above will give the following result

Msg from hello world controller

{{message}}

Explanation of the example

  • "ng-controller" looks for the function with name "HelloWorldController"
  • "<div ng-controller="HelloWorldController">": The div is the scope for the view
  • {{message}}A template. A space holder where we can put data. The scope of the controller can modify
  • function HelloWorldController($scope) is the definition of the function HelloWorldController

Example2

                <div ng-controller="EmployeeController">
                <h2>Result from EmployeeController</h2>
                <table class="table">
                    <tr ng-repeat="employee in employees">
                        <td>{{employee.firstName}}</td>
                        <td>{{employee.lastName}}</td>
                        <td>{{employee.salary}}</td>
                    </tr>
                </table>
                </div>

                <script>
                    function EmployeeController($scope) {
                        var alice = {
                            firstName: "Alice",
                            lastName: "Wonderland",
                            salary: 123321
                        };

                        var bob = {
                            firstName: "Bob",
                            lastName: "Marley",
                            salary: 234234
                        };

                        var charlie = {
                            firstName: "Charlie",
                            lastName: "Garcia",
                            salary: 543253
                        }

                        var employees = [alice, bob, charlie]
                        $scope.employees = employees;
                    }
                </<script>

The code above will give the following result

Result from EmployeeController

{{employee.firstName}} {{employee.lastName}} {{employee.salary}}
