Database Overview

1. Table List

  1. AuthUser: The basic information of all registered users
  2. memberPhoto: Information table of photos that are in members' galleries.
  3. memberSheetMusic: Information table of Sheet Musics that are in members' collections.
  4. userLikeImg: Relationships of which user likes which photo
  5. userLikeSm: Relationships of which user likes which sheet music
  6. sheetMusicComment: Each comment for each sheet music
  7. photoComment: Each comment for each photo
  8. follow: the following relationships between each pair of users.
