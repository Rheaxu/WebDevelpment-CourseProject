Some notes about Syntax

Here are some of my notes about syntax and grammar

HTML

  • div: is very generic, it vertically and logically divides the page to 12 colums
  • radio box: with the same name, radios will be mutually exclusive; with a label as id, clicking on the label of radio will also select the radio

CSS

  • padding: is the distance between actual text and boarder

Javascript

  • JQuery can be abbreviated as $
  • $(“div”).hide(): the div in style will be hidden
  • $(“div”).css(“background-color”,”green”): override the color
  • val padding = $(“input#padding”).val(): grab an input whose id is padding
