Experiment: JavaScript Popup Box

Sometimes, we need to pop up some informations in the browser. The Javascript alert box is really helpful for this

Details of the experiments:

1. Alert Box

Click the following button, you will see a popup alert box with message "Hi, I am an alert box!!!" After you click the "yes" button, it will disappear.

The code for the button is:

<button class="btn btn-info" onclick="popup()">Get Alert Info</button>

<script>
    function popup() {
        window.alert("Hi, I am an alert box!!!");
    }
</script>

2. Confirm Box

Sometiems, you want the user to confirm some information. So instead of just give the "OK" choice, you'd rather let them choose yes/no. The Confirm box is very helpful for this.

The code for the button is:

<button class="btn btn-info" onclick="myConfirm()">Get confirm info
        
<script>
    function myConfirm() {
        var x;
        if (confirm("Choose Ok or Cancel") == true) {
            x = "You pressed OK!"
        } else {
            x = "You pressed Cancel!"
        }
        document.getElementById('confirm_info').innerHTML = x;
    }
</script>
