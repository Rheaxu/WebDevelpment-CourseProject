Experiment: HTML List

HTML list is frequently used. In this experiment, I explored a little bit on this element.

There are three kinds of HTML lists.

1. Unordered HTML List
An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles).

  • Coffee
  • Tea
  • Milk

The code for the above list is:

<ul>    
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ul>

2. Ordered HTML List
An unordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers.

  1. Coffee
  2. Tea
  3. Milk

The code for the above list is:

<ol>    
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ol>
3. HTML Descriptionn List
A description list, is a list of terms, with a description of each term. The <dl> tag defines a description list.
The <dt> tag defines the term (name), and the <dd> tag defines the data (description). </p>
Coffee
- black hot drink
Tea
- green hot drink
Milk
- while cold drink

The code for the above list is:

<dl>
    <dt>Coffee</dt>
    <dd>- black hot drink</dd>
    <dt>Tea</dt>
    <dd>- green hot drink</dd>
    <dt>Mil</dt>
    <dd>- while cold drink</dd>
</dl>
