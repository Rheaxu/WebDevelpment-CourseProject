Photo Detaill Page

Only members have access to this page by logging in.

1. How to access to the photo detail page

This has been mentioned in the Sheet Music Collection and Pohoto Gallery of all members Page, My Photo Gallery Page and Post Photo page

2. Basic information of the photo

1. The basic information includes: Title, Author (the member who put this sheet music in his/her collection), Create Time, Description and count of likes.

2. You can "Like" or "Unlike" this photo on this page

3. Go to the author's profile by clicking the author's name

3. Members who like this photo

At the bottom information, you can see all the members who like this photo. You can explore their profile by cliking their names.

This information is stored in the table userLikeSm. I used the following code to get it:

var likes = (from s in context.userLikeImgs where s.photoId == imgId select s).Count();
likeCountLabel.Text = "Likes: " + likes.ToString();
var likeUsersList = (from s in context.userLikeImgs where s.photoId == imgId select s).ToList();
likeUserRepeater.DataSource = likeUsersList;
likeUserRepeater.DataBind();

4. Add to my Gallery

By clicking the "Add to my Gallery" button, you will be navigated to post photo page to add this work to your gallery.

See Documentation of Post Sheet Music and Post Photo Page

5. Comments

All comments of this photo are listed at the bottom. You also can add your own comments.

By clicking the commentor's name, you can go to his/her profile.

The comments are stored in table photoComments, I used the following code to get them:

List<photocomment> pCmntObjList = (from s in context.photoComments where s.pId == imgId orderby s.createTime descending select s).ToList();
CmntCountLabel.Text = pCmntObjList.Count() + " comments for this photo";
cmntRepeater.DataSource = pCmntObjList;
cmntRepeater.DataBind();
