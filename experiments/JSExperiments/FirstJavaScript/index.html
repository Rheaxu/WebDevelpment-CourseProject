JavaScript Can Change HTML
The HTML DOM (the Document Object Model) is the standard for accessing HTML elements. Using the DOM, JavaScript can change and manipulate HTML.
You can use JavaScript to:
Change HTML content
Change HTML attributes
Change HTML styles
