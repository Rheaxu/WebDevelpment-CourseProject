Experiment: HTML Attributes

I. Why HTML attributes

Attributes provide additional information about HTML elements.

  • HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes come in name/value pairs like: name="value"

II. Examples and explains

In this experiment, we will provide 3 examples to demonstrate HTML attributes.

1. "href" for tag a

The "HERE" is a link that directs your to w3school home page in a new page. Because we set its attribute "href" and "target" attribute as follow:

<a href="http://www.w3schools.com" target="_blank">This is a link</a>

The "HERE" is a link that directs your to Northeastern University home page in a new page. Because we set its attribute "href" and "target" attribute as follow: 

<a href="http://www.northeastern.edu" target="_blank">This is a link</a>

2. "width" and "height" for img tag

The following two pictures are the same picture. They just have different value for attribute "width" and "height"

<img src="http://bighdwallpapers.com/thumbs/happy-smile-facebook-cover-t2.jpg" width="50" height="50" />
        
<img src="http://bighdwallpapers.com/thumbs/happy-smile-facebook-cover-t2.jpg" width="200" height="200" />

3. "Class" for button tag

"Class" is an attribute that help you to paint your element. You can use classes you defined in the css file, or that you defined in the <style></style> tag pair.

The following two buttons are with different classes:

<button class="btn btn-danger">buttonn</button>           
             
<button class="btn btn-info">button</button>
